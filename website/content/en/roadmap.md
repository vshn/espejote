# Roadmap

## Working on

### Web page, documentation scrub, and more examples

We are currently working on improving the documentation and adding more annotated examples to showcase Espejote's capabilities.
Stay tuned for updates!

## Planned features

### API builder

A framework to easily build custom resource definitions using Espejote's core functionalities.
This helps position Espejote as a foundation for building simple operators.

### WASM plugins

Support for WebAssembly plugins to extend Espejote's capabilities beyond the Kubernetes API.
